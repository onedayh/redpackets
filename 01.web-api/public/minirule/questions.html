红包拆开后，红包金额进入用户在芝麻红包小程序的钱包中，可通过在【我的】-【钱包】功能中提取您所领取的金额（可累计提取，每次提取的金额必须大于1元），提取后，资金将转入您的微信零钱中，用户可进入微信零钱明细中查询。

红包创建后，如果非公开到芝麻广场（默认不公开），则您的红包仅可以通过您自己分享的方式发出，其他非您传播用户无法进入领取；如果公开到芝麻广场，则红包可以被平台所有玩家抢，根据您自己的意愿，可设置该选项。

此需求需联系芝麻客服，由客服人员在后台为您设置金额。

此业务需联系芝麻客服，同时公众号只能是认证的订阅号或服务号，您需提供相关信息，由客服人员在后台为您开通红包领取限制；非公众号粉丝无法领取红包。

此业务需联系芝麻客服，需提供导流的小程序相关信息，由客服人员在后台为您开通领取导流。用户领取红包后，引导用户跳转到您的小程序。

红包创建后，联系芝麻客服，由客服人员在后台为您设置时间。

红包创建后，24小时候，如没有领取完毕，红包主可进入小程序 【我的】-【记录】，找到所发的红包，点击【撤回红包】按钮，红包剩余金额将进入零钱，红包主可通过【余额提现】方式提回至个人微信零钱。

芝麻红包在您创建好红包后，提供了两种方式支持您分享红包。

a) 海报分享：创建好红包后，会马上引导您进入红包海报，您可以保存海报图片到手机相册，您可以选择发送图片给好友或发微信朋友圈时从相册中选择此红包海报，您的好友看到图片后可在微信中直接”识别图片小程序码”进入您的红包游戏中领取。

b) 直接分享：您也可以选择分享您的红包链接，直接在您微信好友通讯录中选择需要发送的群或个人，小程序将直接分享至目的位置，收到的用户可直接点击即可进入红包领取。
